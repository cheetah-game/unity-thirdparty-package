## Registry
Add config to UnityProject/Packages/manifest.json
```json
  "scopedRegistries": [
    {
      "name": "cheetah",
      "url": "https://registry.cheetah.games/",
      "scopes": [
        "games.cheetah.unity"
      ]
    }
  ]
```
